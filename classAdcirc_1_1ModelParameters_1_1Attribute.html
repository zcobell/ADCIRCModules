Detailed Description

Class handles nodal attributes, i.e. an ADCIRC unit 13 file.

Author
Zachary Cobell
Copyright
Copyright 2018 Zachary Cobell. All Rights Reserved. This project is released under the terms of the GNU General Public License v3

The Attribute class is designed to handle nodal attributes of variable size (i.e. scalars like friction or 12-parametered values such as directional wind reduction.
